# Music App Frontend

Aplicacion web de musica desarrollada con React.js que consume una Api Rest propia.

# Tecnologias

React
Css
Fetch Api

# Requisitos 

Node.js
npm

# Instalacion y ejecucion

1-Clonar el repositorio: https://github.com/Lud0110/Music-App
2-Entrar a la carpeta: music-app-frontend
3-Instalar dependencias: nm install
4-Crear un archivo .env en la raiz: REACT_APP_API_URL=http://localhost:5000/api
5-Correr el proyecto con npm start
6-Abrir el navegador en: http://localhost:3000

# Funcionalidades

Listado de canciones desde la Api
Filtro por genero musical
Seccion de artistas destacados
Login y registro de usuarios
Diseño responsivo

# Estructura el proyecto

src/
    components---Componentes reutilizables
    pages---Paginas principales
    services---Llamadas a la Api
    styles---Archivos css
    hooks---Custom hooks
